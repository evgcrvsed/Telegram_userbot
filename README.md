# Telegram Userbot для управления сервером

**Userbot на Telethon**, который позволяет управлять VPS-сервером через Telegram, а также запускать парсеры и другие полезные фичи.

## ✨ Возможности

- Запуск bash-скриптов на удалённом сервере
- Удобное добавление новых команд и модулей
- Админ-панель на FastAPI
- Хранение данных в JSON-файлах

## 🛠 Стек технологий

- **Python** 3.12
- **Telethon** — основа userbot
- **FastAPI** — веб-админка
- JSON — хранение настроек и данных
